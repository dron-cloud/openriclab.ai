# Dara Ron Research Website

## Files
- `index.html` — homepage content
- `styles.css` — responsive design and visual styling
- `script.js` — mobile navigation and scroll animations

## How to open
Double-click `index.html` to preview the site locally in a browser.

## Before publishing
1. Replace the portrait placeholder with your professional photo.
2. Update the email address in the Contact section.
3. Add your CV file and update the CV link.
4. Replace placeholder publication and project links.
5. Update publication counts and metrics if needed.

## Add a profile photo
Replace the `<div class="portrait-placeholder">...</div>` block in `index.html` with:

```html
<img class="portrait-placeholder" src="images/dara-ron.jpg" alt="Dr. Dara Ron">
```

Then create an `images` folder and place your photo there as `dara-ron.jpg`.

## Publish free
You can deploy this site using GitHub Pages, Netlify, or Vercel.
